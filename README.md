# Pavan Septic Tank Cleaning

A professional, high-performance website for Pavan Septic Tank Cleaning services in Visakhapatnam.

## Features
- Fully responsive, modern UI with a unique glassmorphism design system
- Built with React and Vite
- Interactive Service Coverage Google Map
- High-converting WhatsApp-integrated contact form

## Setup
1. `npm install`
2. `npm run dev` to start the development server
3. `npm run build` to build for production
